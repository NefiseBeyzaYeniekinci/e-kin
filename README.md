# 📱 E-Kin Mobil Uygulaması

E-Kin, Android platformu için geliştirilmiş modüler yapıda bir mobil uygulamadır. Proje, modern Android geliştirme ilkelerine uygun olarak yapılandırılmıştır.

---

## 🚀 Özellikler

- Modüler mimari (app, core, feature katmanları)
- Gradle tabanlı proje yönetimi
- Modern Android API uyumluluğu
- Kolay geliştirilebilir ve ölçeklenebilir yapı

---

## 🛠️ Kullanılan Teknolojiler

- **Android Studio**
- **Kotlin / Java**  _(Proje diline göre güncellenebilir)_
- **Gradle Build System**
- **MVVM / Clean Architecture** _(Varsayılan — netleştirebiliriz)_

> Kullanılan ek kütüphaneler varsa ekleyebilirim (Retrofit, Room, Firebase vs.)
